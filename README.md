# 文字人生 (Text Life)

一款充满黑色幽默和随机性的文字选择生存游戏

## 游戏简介

在这个游戏中，你将体验一个充满意外和荒诞的人生。从出生到死亡，每个选择都可能改变你的命运。你能活到多少岁？

### 特色

- **极致的随机性**：出生背景、属性、事件全部随机生成
- **100+ 事件**：涵盖人生各个阶段的有趣事件
- **黑色幽默**：各种荒诞离奇的死亡方式
- **属性影响**：健康、智力、运气会影响事件结果
- **终端风格 UI**：二进制高级感的黑客风格界面
- **排行榜系统**：记录你的前 3 次最长寿命

## 游戏玩法

1. **开始游戏**：点击"开始新人生"按钮
2. **查看角色**：系统会随机生成你的姓名、性别、出生背景和初始属性
3. **做出选择**：阅读事件描述，从两个选项中做出选择
4. **观察结果**：你的选择会影响属性和年龄
5. **生存挑战**：尽量活得更久！

## 属性说明

- **健康 (Health)**：影响疾病抵抗力，降到 0 会死亡
- **智力 (Intelligence)**：影响选择判断，智力高能识别危险选项
- **运气 (Luck)**：影响随机事件结果，运气高能避免突发死亡
- **魅力 (Charm)**：影响社交相关事件

## 事件类型

### 按年龄段分类
- **0-3 岁**：婴儿期事件
- **4-12 岁**：儿童期事件
- **13-18 岁**：青少年期事件
- **19-30 岁**：青年期事件
- **31-50 岁**：中年期事件
- **51-122 岁**：老年期事件

### 特殊事件
- **突发死亡**：低概率随机触发，受运气影响
- **荒诞事件**：充满黑色幽默的离奇事件
- **两难选择**：有时两个选项都很危险！

## 死亡方式示例

游戏中包含各种有趣的死因，例如：
- 被天降花盆砸死
- 因为舔冬天的金属栏杆而死
- 蹲厕所时间太长猝死
- 忍喷嚏导致血管爆裂
- 泡面忘记加水引发火灾
- 等等...（100+ 种不同的死法）

## 技术栈

- **纯前端**：HTML + CSS + JavaScript
- **无依赖**：不需要任何第三方库
- **本地存储**：使用 localStorage 保存排行榜

## 适配抖音小游戏

游戏设计为纯前端应用，可以轻松适配抖音小游戏平台：

1. 使用 Cocos Creator 导入 HTML5 项目
2. 或者直接打包为字节小程序
3. 接入抖音 API（登录、分享、广告等）
